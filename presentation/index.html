How It Started

January 2025. Factory tour. Supervisor mutters: “Wish those cameras told me who’s missing hard-hats.”
Challenge accepted.

  • Week 1 → Roboflow PoC — 37 % mAP (ouch).
  • Week 3 → first on-site test — posters triggered false detections 🤦‍♂️.
  • Pivot → collect & label our own footage + crank up synthetic augments.

Journey in Milestones

  1. Feb 10: PoC to advisors → green-light.
  2. Mar 01: 18 k frames recorded across 3 partner sites.
  3. Mar 20: Unity synthetic pipeline → +120 k PPE images.
  4. Apr 08: Switched YOLOv5-s → YOLOv8-s (↑ 2.4 mAP).
  5. Apr 25: Distilled & pruned (−40 % params, same accuracy).
  6. May 12: Jetson Orin Nano demo @ 31 FPS.

What We Learned 🙌

  • 🎨 Lighting beats algorithms → heavy domain randomisation.
  • 📝 Annotation fatigue is real → cap at 200 imgs/day + QA panel.
  • 🚀 Edge > cloud for latency (−180 ms).
  • 🧪 Synthetic vest colours ↑ recall +5 pp.

How It Can Be Improved 🔭

  1. Multi-camera fusion (ID tracking across views).
  2. Pose-based unsafe actions (bending, climbing, phone use).
  3. INT4 + sparsity → Raspberry Pi 5.
  4. Active-learning loop for auto-label of false-positives.
