OshaSight

A live, always-on PPE & smoke monitor built with YOLOv8. Detects mask, gloves, helmet, goggles, hi-viz vest and smoking in real time (<50 ms/frame).

View Demo Slides

Key Features

  • Six independent YOLOv8 detectors—one per class (Mask, Gloves, Helmet, Goggles, Vest, Smoke)
  • Real-time overlays & alerts via a PySimpleGUI dashboard
  • 100 FPS on GTX 1660 Ti or 31 FPS on Jetson Orin Nano (≈ 5 W)
  • Signed CSV log for OSHA-style audits + optional audio/visual alarms
  • Fully open-source: VOC→YOLO converter + train_*.py scripts
  • See Setup for training, weights download and Edge deployment

About OshaSight

Developed for Sapienza’s “Computer Vision & NLP” lab (Spring 2025). We converted public datasets into YOLO format, trained each class on ultralytics’ YOLOv8-s or -n backbones (50 epochs, batch 8, imgsz 640), then distilled our best model for edge deployment.

Datasets & Links
